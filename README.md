# BATMAN - Battery Manager CLI Tool

BATMAN is a lightweight Linux utility to manage battery conservation mode via CLI.

## Features
- View battery status
- Enable/Disable conservation mode
- Set charge limit

## Installation
```bash
chmod +x install.sh
./install.sh
```

## Usage
```bash
batman
```

## Uninstallation
```bash
./uninstall.sh
```
